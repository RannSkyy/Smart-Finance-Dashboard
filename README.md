SmartFin — Smart Finance Dashboard

A modern financial dashboard with a futuristic dark premium design built to help users manage and visualize their financial activity more effectively.

✨ Main Features
- Real-time finance statistics
- Total balance, income, expenses, and savings cards
- Interactive monthly cashflow chart
- Financial score analysis
- AI insight section
- Modern sidebar navigation
- Transaction search bar
- Responsive design (Desktop & Mobile)
- Premium dark mode UI
- Smooth animations & hover effects
  
🛠️ Technologies Used
-HTML5
-CSS3
-JavaScript
-Chart.js
-Remix Icon / CDN Icons

📊 Dashboard Components
- Dashboard Overview
- Financial Statistics
- Cashflow Analytics
- Financial Health Score
- Transactions
- Analytics
- Budget Management
- Account Settings
